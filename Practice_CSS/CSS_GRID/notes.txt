Grid vs Flexbox

GRID is a 2 Dimensional system          - we have control of how we layed out horizontally or vertically
FLEXBOX is a 1 Dimensional system       - control on how you lay out elements in column or row (one or the other)


Grid Line - a line dividing grid column or row
Grid Track - a full column or row
Grid Area - total space in collection of cells
Grid Cell - a single cell
